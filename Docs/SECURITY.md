# Security Policy

## Supported Versions

Currently, only the latest release of Tailscale-Headscale Client Pro receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 5.0.x   | :white_check_mark: |
| < 5.0   | :x:                |

## Reporting a Vulnerability

We take the security of our application very seriously. If you discover a security vulnerability within this project, please do NOT file a public issue. 

Instead, please send an email to the project maintainers or report it privately via GitHub Security Advisories. 

We will acknowledge your report within 48 hours and provide a timeline for the fix and disclosure.
